# Installation Steps for JWT Auth & Roles

Follow these steps in order to set up the authentication and authorization system.

## Step 1: Install Required Packages

```bash
cd api

# Install JWT Auth
composer require tymon/jwt-auth:^1.0

# Install Spatie Permission (should already be done, but just in case)
composer require spatie/laravel-permission:^4.7

# Install Intervention Image (for future media handling)
composer require intervention/image:^2.7
```

## Step 2: Publish Configurations

```bash
# Publish JWT config
php artisan vendor:publish --provider="Tymon\JWTAuth\Providers\LaravelServiceProvider"

# Publish Spatie Permission config (if not already done)
php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"
```

## Step 3: Generate JWT Secret

```bash
php artisan jwt:secret
```

This will automatically add `JWT_SECRET` to your `.env` file.

## Step 4: Update .env File

Make sure your `api/.env` includes:

```env
DB_DATABASE=MamoKacha
DB_USERNAME=root
DB_PASSWORD=your_password_here

JWT_SECRET=<auto-generated>
JWT_TTL=60
JWT_REFRESH_TTL=20160
```

## Step 5: Run Migrations

```bash
# Run Spatie Permission migrations (if not already run)
php artisan migrate

# Or run all migrations
php artisan migrate
```

## Step 6: Seed Roles and Users

```bash
php artisan db:seed
```

This will create:
- Roles: SuperAdmin, Editor, Author, Viewer
- Permissions for all modules
- SuperAdmin user: `admin@example.com` / `Admin@12345`

## Step 7: Test the API

### Test Health Endpoint
```bash
curl http://localhost:8000/api/v1/health
```

### Test Login
```bash
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@example.com","password":"Admin@12345"}'
```

### Test Protected Endpoint (use token from login response)
```bash
curl -X GET http://localhost:8000/api/v1/auth/me \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

### Test Abilities Endpoint
```bash
curl -X GET http://localhost:8000/api/v1/me/abilities \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

## Troubleshooting

### Issue: "Class 'Tymon\JWTAuth\Facades\JWTAuth' not found"
**Solution:** Run `composer dump-autoload` and verify JWT package is installed.

### Issue: "JWT secret not set"
**Solution:** Run `php artisan jwt:secret`

### Issue: "Roles table not found"
**Solution:** Run `php artisan migrate` to create Spatie Permission tables.

### Issue: "No query results for model [Role]"
**Solution:** Run `php artisan db:seed` to seed roles and permissions.
