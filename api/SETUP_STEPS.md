# Setup Steps - Run These Commands

## Step 1: Install Dependencies

```bash
cd api
composer install
```

This will install all Laravel and required packages (JWT Auth, Spatie Permission, etc.)

## Step 2: Configure Environment

Make sure your `api/.env` file has the correct database credentials:

```env
DB_DATABASE=MamoKacha
DB_USERNAME=root
DB_PASSWORD=your_password_here
```

Also ensure these are set (they should be auto-generated):
```env
APP_KEY=base64:... (generated by php artisan key:generate)
JWT_SECRET=... (generated by php artisan jwt:secret)
```

## Step 3: Run Migrations & Seed Data

```bash
# Run migrations to create all tables
php artisan migrate

# Seed roles, permissions, and SuperAdmin user
php artisan db:seed
```

After seeding, you'll have:
- SuperAdmin user: `admin@example.com` / `Admin@12345`
- Roles: SuperAdmin, Editor, Author, Viewer
- Permissions for all modules

## Step 4: Test the Setup

```bash
# Start Laravel server
php artisan serve
```

In another terminal, test the API:

```bash
# Test health endpoint
curl http://localhost:8000/api/v1/health

# Test login
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d "{\"email\":\"admin@example.com\",\"password\":\"Admin@12345\"}"
```

## Troubleshooting

### If composer install fails:
- Make sure Composer is installed
- Check PHP version (needs PHP 7.4+)
- Try: `composer install --no-interaction`

### If migrations fail:
- Check database connection in `.env`
- Make sure database `MamoKacha` exists
- Create database: `CREATE DATABASE MamoKacha;`

### If JWT secret is missing:
```bash
php artisan jwt:secret
```

### If APP_KEY is missing:
```bash
php artisan key:generate
```
